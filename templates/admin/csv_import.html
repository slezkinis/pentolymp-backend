Формат CSV файла

CSV файл должен содержать следующие колонки:

  • name — Название задачи
  • description — Условие задачи (HTML формат)
  • answer — Правильный ответ
  • subject — Предмет
  • topic — Тема
  • difficulty_level — Уровень сложности (Easy / Medium / Hard)
  • tip — Подсказка (необязательно)

Пример:

name,description,answer,subject,topic,difficulty_level,tip
"Простая задача","Решите: 2+2","4","Математика","Арифметика","Easy","Сложите два числа"
"Сложная задача","Найдите производную x²","2x","Математика","Матанализ","Hard","Используйте правило степени"
